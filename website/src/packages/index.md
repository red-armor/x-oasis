## Featured Packages

### async-call-rpc

Bidirectional RPC protocol framework with pluggable middleware support for various transport layers.

- **Learn more**: [async-call-rpc Documentation](/packages/async/async-call-rpc/)
- **Transports**: MessagePort, WebSocket, IPC, child_process, Electron
- **Features**: Middleware pipeline, offline queueing, context injection, subscriptions
