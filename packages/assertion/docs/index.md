# Assertion Packages

Type checking and validation utilities for safe value inspection

## Packages in this Category

- **[@x-oasis/is-empty](/packages/assertion/is-empty/)** - Check if a value is empty (null, undefined, empty string, etc.)
- **[@x-oasis/is-null](/packages/assertion/is-null/)** - Check if a value is null
- **[@x-oasis/is-object](/packages/assertion/is-object/)** - Check if a value is a plain object
- **[@x-oasis/is-function](/packages/assertion/is-function/)** - Check if a value is a function
- **[@x-oasis/is-promise](/packages/assertion/is-promise/)** - Check if a value is a Promise
- **[@x-oasis/is-class](/packages/assertion/is-class/)** - Check if a value is a class or class instance
- **[@x-oasis/is-primitive](/packages/assertion/is-primitive/)** - Check if a value is a primitive type
- **[@x-oasis/is-primitive-empty](/packages/assertion/is-primitive-empty/)** - Check if a primitive value is empty
- **[@x-oasis/is-nan](/packages/assertion/is-nan/)** - Check if a value is NaN
- **[@x-oasis/is-ascii](/packages/assertion/is-ascii/)** - Check if a string contains only ASCII characters
