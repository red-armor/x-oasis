🔌 WebSocket RPC 客户端测试

状态: 正在连接 WebSocket 服务器...

📋 可用的服务器方法

  • server.echo(x) - 回显传入的参数
  • server.now() - 返回当前时间戳
  • server.add(a, b) - 计算两个数字的和

💡 在浏览器控制台中测试

打开浏览器开发者工具 (F12)，在控制台中输入以下命令：

// 测试 echo 方法
await server.echo("Hello!")

// 测试 now 方法
await server.now()

// 测试 add 方法
await server.add(10, 20)

// 测试复杂对象
await server.echo({name: "test", value: 42})

提示: 也可以使用 window.remote 代替 server

📝 检查清单

  • ✅ 确保 WebSocket 服务器正在运行 (node node.websocket.server.js)
  • ✅ 打开浏览器控制台查看自动测试结果
  • ✅ 在控制台中手动测试 RPC 调用
  • ✅ 检查网络标签页中的 WebSocket 连接状态
