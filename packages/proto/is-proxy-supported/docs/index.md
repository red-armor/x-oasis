# @x-oasis/is-proxy-supported

Check if the runtime environment supports `Proxy`.

## Installation

```bash
npm install @x-oasis/is-proxy-supported
```

## Quick Start

```typescript
import isProxySupported from '@x-oasis/is-proxy-supported';

if (isProxySupported()) {
  // Proxy is available
}
```

## Key Features

- Zero dependencies
- TypeScript support
- Works in browsers and Node.js

## API Reference

### `isProxySupported(): boolean`

Returns `true` if the runtime supports `Proxy`, `false` otherwise.
