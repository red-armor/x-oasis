# @x-oasis/format-date

## Installation

```bash
$ npm i @x-oasis/format-date
```

## How to use

```typescript
import { format } from '@x-oasis/format-date'

format('yyyy/MM/dd HH:mm:ss', new Date())
```

Supported tokens: `yyyy`, `MM`, `dd`, `HH`, `mm`, `ss`, `sss`.

## How to run test

```bash
$ pnpm test
```
